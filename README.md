# BarpVN

BarpVN is a game engine for visual novel games.

Planned features:
- Linux support
- Windows support? (If I feel like it once I finish linux. Cross-platform is hard 3:)
- JSON-based game logic ([Concept](https://gist.github.com/DessieOwO/29aeb05b192987e9b4e225a2ddbc03fb))
- Completely customizable styling
- Manual/auto saving and loading
- Persistent settings

Potential future projects:
- Game editor
- Web transpiler (Packages game to be played in a browser)
